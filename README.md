# nsc-cli

## INSTALL

```bash
npm i @lad-tech/nsc-cli

```

## USAGE

```bash 
npx nsc-cli --schema ./services/myService/service.json     
```

```bash

npx nsc-cli -h 

Usage: nsc-cli [options]

Генерация нового сервиса по json schema

Options:
  --schema  <path>  путь до схемы
  -h, --help        display help for command




```
